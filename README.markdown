# This is a demonstration of how you can use Cucumber with Harmony.

You should do the following for some fun times.

    $ gem install stackdeck
    $ gem install envjs
    $ gem install johnson -v "2.0.0.pre3"
    $ gem install harmony
    $ gem install cucumber
    $ cucumber
